# 📊 Power BI Sales Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard that analyzes sales performance, profit, and regional trends.

## 📁 Files Included
- dashboard.pbix → Power BI file
- dataset.csv → Dataset used for analysis
- screenshot.png → Dashboard preview

## 📸 Dashboard Preview
![Dashboard](screenshot.png)

## 🛠 Tools Used
- Power BI Desktop
- Microsoft Excel

## 📊 Key Insights
- Sales increased in recent months
- West region has highest profit
- Technology category performed best

## 🎯 Purpose
To visualize business data and generate actionable insights using Power BI.
